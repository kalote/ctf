# Starter - Don't look at the source code

## Local install

`npm install`
`npm start`

## Docker install

```bash
docker build -t ctf_starter .
docker run --name ctf_starter -d -p 8080:8080 -t ctf_starter:latest
```

## Endpoint

open your browser on http://localhost:8080
